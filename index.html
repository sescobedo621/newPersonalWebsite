SuperWiki

SuperWiki is a superhero Wikipedia site and fan forum. Users can add their favorite superheros and villains, update any incorrect information, and talk with fellow fans about their favorite superhero. It is made with Java, Spring MVC, JPA, MYSQl for the database, with HTML, CSS, and Bootstrap in the front-end.

getFit will keep track of daily exercise routines. Users can follow of all of their previous exercises. They can also see their average daily steps. getFit was made with Java REST and MYSQL database for the backend. It uses JavaScript DOM, HTML, and CSS for the front-end.

Whack-A-Mole

Take your frustrations out with Whack-A-Mole. Users have 30 seconds to hit as many moles as possible, but more join in the fun as the timer winds down. List your name and see if you can beat the high score. Whack-A-Mole was made with Java REST and MYSQL database for the backend to keep track of scores. The game itself. The front-end was made with JavaScript, HTML and CSS.

Meta Clothing

Meta Clothing is an e-commerce web application. Users can create an account, login, and browse items. They can also add items to their shopping cart, as well as edit their address. Meta Clothing is made with Java RESTful API, Spring, JPA, MYSQL database. It is connected to a Node application on the client side. The front-end was created with HTML and CSS, and Bootstrap.
